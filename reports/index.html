Algorithm PCA — Principal Component Analysis
An unsupervised learning method. It was trained only on normal logs — it has never seen attacks or failures. It learned the mathematical fingerprint of a normal session: which events appear, how often, and in what combinations. When it encounters a new session, it measures how much that session differs from its learned fingerprint. A large difference means the session is anomalous.
